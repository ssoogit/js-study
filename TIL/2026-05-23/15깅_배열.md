## 배열(Array)이란?
순서가 있는 요소들의 집합, 여러 개의 데이터들이 모여있는 리스트 

## 배열 생성 방법
객체와 동일하게 **생성자 함수나 리터럴을 사용**해서 생성할 수 있다. 

### 1. 생성자 함수
* Array() 라는 생성자 함수를 사용. `Array()`

* 배열 생성자 함수 array는 new 라는 키워드와 함께 작성해야함. `new Array()`

* `let arr2 = new Array(3);`
  * 숫자 3이라는 값이 배열 요소에 할당되는 것이 아니라, 비어있는 3개의 값이 할당되어 길이가 3인 비어있는 배열이 출력된다.

#### 배열 생성자의 값 할당 방법
* 배열에 특정 값을 할당: 생성자 함수의 괄호 안에 여러 개의 값을 작성 `new Array(1, 2, 3);`
* 배열에 원하는 크기의 공간을 할당: 원하는 공간의 크기를 괄호에 작성 `new Array(3);`

### 2. 리터럴
* 중괄호가 아닌 대괄호를 사용해 배열을 생성

### 생성자 함수와 리터럴의 차이점
* `let arr2 = [3];` 는 생성자 함수로 생성된 배열(길이가 3인 배열 생성)과 달리, 숫자 3이라는 값이 배열 요소에 할당되어 [3]이 출력된다.

## 배열 요소 조작(접근) 방법
데이터 구분 키 값 존재 x, 인덱스를 통해 접근 (배열의 인덱스는 0번부터 시작)

## 배열 요소 추가 및 수정, 삭제 방법 (예제 코드 참고)
### 배열 요소 추가 메서드
* push() : 마지막 인덱스에 추가
* unshift() : 첫번째 인덱스에 추가

### 배열 요소 수정 (2가지)
* 직접적인 수정: `animal = ['cat', 'rabbit', 'hamster'];`
* 해당 인덱스 접근: `animal[2] = 'parrot';`
* **const로 선언할 경우, 직접적인 수정은 불가능하다. 하지만, 해당 인덱스에 접근해서 넣는 건 가능하다.**
  * 자바스크립트에서 배열은 객체로 분류되는 자료형이기 때문.

### 배열 요소 삭제 메서드
* pop() : 마지막 요소 삭제
* shift() : 첫번째 요소 삭제
* splice(start, deleteCount) : 특정 인덱스의 요소 삭제
  * start: 배열 요소의 특정 인덱스
  * deleteCount: 삭제할 요소의 개수

### 배열의 길이 측정 방법
* length 프로퍼티 사용

## 배열과 객체의 차이점
### 생성 방법
**1. 생성자 함수**
* 배열: `let arr = new Array();`
* 객체 : `let obj = new Object();`

**2. 리터럴**
* 배열: 대괄호 사용 `let arr = [];`
* 객체: 중괄호 사용 `let obj = {}; `

### 접근 방법
* 객체: 각 키 값이 존재 o, 키 값을 통해 접근
* 배열: 각 키 값이 존재 x, 인덱스를 통해 접근

## 배열과 객체의 공통점
* 객체 프로퍼티의 값과 배열의 요소 둘 다 자료형에 상관 없이 작성 가능하다.
