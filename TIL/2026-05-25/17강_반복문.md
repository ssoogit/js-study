## 반복문의 종류
* for문
* while문

### 1.for문
조건에 따라 일정 횟수만큼 같은 코드를 반복 실행한다.

`for(초기화식; 조건식; 증감식) {}` - 조건식이 만족할 때까지만 반복

### 2. while문
특정 조건이 참이라면 해당 코드를 반복 수행

`while(조건식){ }` - {} 내부에 증감식을 작성함 (무한 굴레에 빠지지 않게 주의)


## 반복문(for문)을 사용하여 배열의 모든 요소를 순회
* 배열의 모든 요소에 접근해야할 때 유용하게 사용 가능

## 반복문(for문)을 활용해 객체의 프로퍼티 순회
* 객체의 프로퍼티를 순회하기 위해선 객체를 배열의 형태로 변환해 줘야한다.

### 객체(Object)를 배열(Array)로 변경하는 3가지 방법 + for 문 사용

#### 1. 키값으로만 이루어진 배열로 변환 : Object.keys
* 매개변수로 받은 객체의 키들을 모두 찾아 배열의 형태로 반환하는 객체 메서드
* `Object.keys(객체명)` -> 키값을 이용하여 for문으로 value값 찾기! **

#### 2. value값으로만 이루어진 배열로 변환 : Object.values
* 매개변수로 객체를 넘기면 해당 객체의 밸류들을 모두 찾아 배열의 형태로 반환하는,,
* for문으로 value값 나열하기!

#### 3. 키와 밸류 쌍의 형태로 배열 반환 : Object.entries
* 키와 밸류 값을 배열에 담아 새로운 배열로 반환해주는 메서드
* 배열 내부를 확인해보면 알 수 있듯이, array값이 담긴다. (내부 출력 결과 ex:`[Array(2), Array(2), Array(2)]` )

## 알아두면 유용한 자바스크립트의 또 다른 반복문들
### 1. for-of 문
* 주로 배열의 모든 요소에 접근해야될 때 사용한다.
* `for (변수 선언 of 배열 이름) { 선언된 변수로 배열 요소에 접근 가능 }`

### 2. for-in 문
* 주로 객체에서 사용, 모든 프로퍼티들을 바로 순회한다.
* `for (변수 선언 in 객체 이름) { 선언한 변수=키값, 객체[선언한 변수] = 밸류 값 }` <- 이걸 사용하면 배열로 변환 과정 없이 순환이 가능하다.
