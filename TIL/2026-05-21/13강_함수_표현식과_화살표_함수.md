## 화살표 함수
화살표 모양을 사용해 선언하는 함수

`const func = () => { ... }`

간단한 식의 경우 중괄호와 return을 생략하여 작성 가능

## 함수 선언식과 함수 표현식
### 함수 선언식
기본 함수 선언 방법

`function func() { ... }`

### 함수 표현식
변수에 함수를 마치 하나의 값처럼 할당

`const func = () => { ... }`

`const func = function () { ... }`

## 선언 방식의 차이점 => 호이스팅

함수 선언식으로 선언했을 경우 => 선언된 것보다 호출이 먼저 이루어져도 문제가 발생하지 않음.

함수 표현식으로 선언했을 경우 => 선언된 것보다 호출이 먼저 이루어지면 호이스팅의 방식처럼 변수의 선언 부분(const hoisted )만 호이스팅 되기 때문에 문제 발생

## 콜백 함수
매개변수로 전달받은 함수

코드의 재사용성을 높이고, 함수의 동작을 보다 유연하게 변경 가능하다.
