# 2026-05-28 | 배열과 객체 구조 분해 할당

## 구조 분해 할당
배열이나 객체의 요소 및 프로퍼티들을 분해해서 그 값들을 각각의 변수에 할당하는 자바스크립트의 표현식
배열 요소의 값을 추출할 때 자주 사용되지만, 두개의 변수 값을 서로 바꿀 때 사용되기도 한다.

### 선언 분리 할당
* 변수 선언을 분리해서 값을 할당하는 방법
  ```
  let a, b;
  [a, b] = [1, 2];
  ```

## 객체의 구조분해 할당
객체의 구조분해 할당은 인덱스를 이용해 인덱스의 순서대로 변수의 값을 할당하는 배열의 구조분해와는 달리, 킷값을 기준으로 객체를 분해해 변수에 할당한다는 차이점이 있다. 

( `let {c3, c1, c2} = colors;` 라고 입력해도 똑같이 할당된다. )
