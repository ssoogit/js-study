# 2026-05-19 | 함수 선언 & Early return pattern

## 함수
- 특정 작업을 수행하기 위해 작성된 재사용 가능한 코드 블록
- 정의 후 필요할 때마다 호출하여 사용할 수 있다.
- 함수 정의 방법에는 **함수 선언식과 함수 표현식, 화살표 함수**가 있다.

## Early return pattern - 함수 패턴 사용
특정 조건에 부합하지 않으면 바로 리턴하는 패턴

else문을 사용하지 않고 **if와 return으로만 코드를 작성**하며 특정 조건문을 만족하면 바로 return하기 때문에, 이후의 조건은 실행되지 않고 반환하여 코드의 구조를 단순하게 만들어주는 패턴이라고 할 수 있다.
